 **hdparm命令** **-->用来检测、显示及设定IDE或SCSI硬盘的参数; 测试硬盘读性能的常用工具**

 ****

 【**hdparm** **命令作用** 】

 1) dd # 它被用来在Linux和类Unix系统下对硬盘设备进行写性能的检测。  
2) hparm # 它用来在基于 Linux的系统上获取或设置硬盘参数，包括测试读性能以及缓存性能等。

 

![][0]

 【 **检查和收集硬件信息的几个命令** 】  
1) lscpu # 用于查询CPU信息  
2) lshw  # 用于显示硬件信息表  
3) hwinfo  # 用于查询硬件信息  
4) lsppci  # 用于列出PCI总线的信息以及连接到PCI总线上的设备信息  
5) lsusb # 用于列出USB总线信息  
6) lsblk # 用于列出块设备的信息  
7) lsscsi  # 用于列出SCSI的设备信息
