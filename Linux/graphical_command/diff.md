 **diff命令** **-->differential的缩写,用于比较文件的差异，以逐行的方式，比较文本文件的异同处.**

 **打补丁例子:**


    diff -urN old/ new/ > my.patch

参数:-u 表示使用统一的格式  
参数:-r 表示递归比较子目录  
参数:-N 表示将不存在的文件当作空文件处理，这样新添加的文件也会出现在patch文件中。  

     patch -p0 < my.patch

参数:-p 表示跳过几级目录  
参数: 0 表示无需跳过
