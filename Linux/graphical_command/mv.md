 **mv命令** **-->move的缩写;Linux中覆盖和重命名的概念几乎是一样的，没有Windows中所谓的rename这一说法;选项可以写在任意位置，可以mv file1 -v file2，也可以mv -v file1 file2，还可以mv file1 file2 -v**

 ****

 **【 mv命令作用 】**

 作用一:文件名或目录更名

 作用二:将文件或目录移动位置
